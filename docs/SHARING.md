# Sharing Checklist

Recommended public-sharing checklist for this repository:

1. Build the latest package:

```bash
./package_release.sh
```

2. Verify the newest package exists in `dist/`.

3. In your repo description or release post, mention:
   - unofficial community patch
   - requires user-installed `dLive Director V2.11`
   - not affiliated with Allen & Heath
   - use at own risk

4. Do not bundle the vendor app.
